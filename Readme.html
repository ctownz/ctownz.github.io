Gage UI, Input Component

Properties, Notes, Info

Component width is 100% of it's container
Add direction="validate" to show error messages. Messages are not customizable
Size: size="small" or size="medium" or size="large"
To change the background color: bg="red"
Text color is inherited.
Border: border="4px solid red"
border-radius: radius="1.5px"
Value: value="John Doe" or for dates, value="1980-12-25"
id and name: id="street", name="address2"
Label: If you want to add a label, label="Email". The label appears above the input and is left aligned.
To eliminate the *:focus { outline: 0; }
